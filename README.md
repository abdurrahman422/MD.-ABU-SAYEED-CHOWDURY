# MASC Portfolio: Md. Abu Sayeed Chowdhury

A high-end, corporate one-page professional portfolio for **Md. Abu Sayeed Chowdhury**, a Senior Operations Leader and Feed Mill Specialist with over 23 years of global excellence.

## 🌟 Overview

This portfolio is designed with a **"Dark Executive"** aesthetic, reflecting the authority, sophistication, and result-oriented nature of a senior operations leader. It showcases a career spanning from the marine engineering shipyards of Singapore to the leading agro-industries of Bangladesh.

## 🛠️ Tech Stack

- **Framework**: [React 18+](https://react.dev/)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/) (Modern Utility-first CSS)
- **Animations**: [Motion](https://motion.dev/) (Framer Motion)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Typography**: [Roboto](https://fonts.google.com/specimen/Roboto) (Professional Sans-serif)

## ✨ Key Features

- **Hero Showcase**: A bold entrance highlighting the 23+ years of expertise with a professional portrait.
- **Executive Summary**: A concise, impactful narrative of leadership and global operations.
- **Core Competencies**: A grid-view of technical and leadership skills with professional iconography.
- **Professional Journey**: An interactive timeline tracing 20+ years of career evolution.
- **Impact Metrics**: Key performance indicators showing scale (48 MT/H, 42,000 MT capacity, 300+ workforce).
- **Technical Proficiency**: Visual bars demonstrating specialized knowledge in technology and ERP systems.
- **Global Footprint**: Highlighting multicultural experience across 24+ countries.
- **Direct Connect**: Professional contact form and integrated WhatsApp floating button for immediate engagement.

## 🚀 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (Latest LTS version recommended)
- npm or yarn

### Installation

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```
